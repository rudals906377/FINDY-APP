# FINDY Logo Assets - RGB(164, 120, 100)

공통 로고 색상: `RGB(164, 120, 100)` / `#A47864`

이 폴더는 완성본 로고 패키지입니다.

## transparent
앱, 웹, Figma 등 배경 위에 자연스럽게 올릴 때 사용하는 투명 PNG입니다.
투명 영역의 RGB는 `(0, 0, 0, 0)`으로 정리되어 갈색 박스로 보이지 않게 처리했습니다.

## white-bg
문서, PPT, 카카오톡 등 투명 배경을 제대로 처리하지 못하는 환경에서 사용하는 흰 배경 PNG입니다.

## 구성
이 폴더에는 앞으로 사용할 완성본 로고만 보관합니다.
`findy_logo_wordmark.png`는 쌍안경 심볼 없이 FINDY 글자만 잘라낸 헤더용 워드마크입니다.
`findy_logo_mark_shell.png`는 눈동자 애니메이션을 위해 눈동자만 제거한 쌍안경 심볼입니다.
