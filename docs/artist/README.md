# FINDY Artist Policy

이 문서는 FINDY 아티스트 계정과 프로필 관리 정책을 정리한 기준 문서입니다.

## 기본 원칙

아티스트 프로필의 핵심 신뢰 정보는 아티스트가 앱에서 바로 수정할 수 없습니다.
수정이 필요한 경우 FINDY 운영팀에 수정 요청을 보내고, 운영팀 컨펌 후 반영하는 방식으로 관리합니다.

## 운영팀 컨펌이 필요한 주요 정보

아래 정보는 사용자 신뢰와 검색 노출, 예약 연결에 직접 영향을 주므로 승인 절차가 필요합니다.

- 아티스트 이름
- 분야
- 지역
- 샵이름

## 승인 자료

아티스트 가입 또는 주요 정보 변경 시 아래 자료를 FINDY 운영팀에 제출합니다.

- 명함
- 인스타그램 아이디
- 네이버 플레이스에 등록된 사진

운영팀은 제출 자료를 기준으로 실제 활동 여부, 샵 정보, 분야, 지역 정보를 확인한 뒤 승인합니다.

## 앱 화면 기준

아티스트 모드의 프로필 관리 화면은 다음 기준을 따릅니다.

- 주요 정보는 읽기 전용으로 표시합니다.
- 주요 정보 옆에는 직접 수정 입력창을 두지 않습니다.
- 변경이 필요한 경우 `프로필 정보 수정 요청` 흐름으로 이동합니다.
- 수정 요청 화면에서는 변경 희망 정보와 사유를 작성하게 합니다.
- 실제 반영은 운영팀 승인 이후 처리합니다.

## 직접 수정 가능 여부

1차 기준은 다음과 같습니다.

- 직접 수정 불가: 아티스트 이름, 분야, 지역, 샵이름
- 추후 정책 결정 필요: 소개 문구, 영업시간, 상담 가능 여부, 인스타그램/포트폴리오 링크

## 현재 화면 표시 기준

승인 전 또는 더미 데이터 단계에서는 실제 특정 아티스트/샵처럼 보이지 않도록 아래처럼 표시합니다.

- 아티스트 이름: `OO 아티스트`
- 샵이름: `OO 샵`

## 이후 개발 메모

프로필 관리 화면을 수정할 때는 이 문서를 기준으로 주요 정보를 일반 입력 필드로 만들지 않습니다.
주요 정보 수정은 별도 요청 페이지 또는 운영팀 검수 플로우로 분리합니다.
