# FINDY2 뷰티 감성어/추천 키워드 사전

FINDY2는 커뮤니티에서 사용자가 남기는 글, 스냅, 비디오, 나의 FINDY 정보를 모아 추후 FINDY 추천/예약 알고리즘을 강화하는 첫 배포 앱입니다. 이 문서는 헤어, 네일아트, 메이크업, 포토, 웨딩, 반영구시술에 사용할 감성어와 선택지를 정리한 기준입니다.

실제 데이터 파일은 `data/beauty_mood_keywords.py`입니다.

## 왜 필요한가

사용자는 “헤어”처럼 딱딱한 카테고리명보다 “세련된 파란색 짧은머리”, “귀여운 젤리 네일”, “센치한 필름톤 프로필”처럼 감성어와 형태를 섞어서 표현합니다. 이 단어를 구조화해두면 앱이 사용자의 취향을 더 잘 이해할 수 있습니다.

- 검색어를 카테고리, 분위기, 색감, 목적, 고민으로 나눠 해석할 수 있습니다.
- 글쓰기, 스냅, 비디오 캡션에 자동 태그를 붙일 수 있습니다.
- 나의 FINDY에 저장한 취향과 커뮤니티 글을 매칭할 수 있습니다.
- 예약 서비스로 확장할 때 사용자의 취향과 샵/디자이너 포트폴리오를 비교할 수 있습니다.
- 신규 사용자가 데이터가 적을 때도 기본 키워드 사전으로 초기 추천을 만들 수 있습니다.
- 이후 실제 클릭, 좋아요, 저장, 시청 완료 데이터를 반영해 가중치를 조정할 수 있습니다.

## 수집 기준

공개 트렌드 자료와 뷰티 콘텐츠에서 반복적으로 등장하는 표현을 기준으로 1차 사전을 만들었습니다. 이 사전은 최종 정답이 아니라 FINDY2에서 데이터가 쌓이기 전까지 사용하는 초기 시드입니다.

참고한 방향은 다음과 같습니다.

- Pinterest Predicts 2026: 다크로맨틱, 뱀파이어, 고스, 쿨블루, 아이스블루 계열
- Vogue/Allure 2026 메이크업 트렌드: 블러립, 애교살, 언더아이 블러셔, 컬러풀/홀로그램, 글로우 피부
- 2026 네일 트렌드: 젤리, 자석젤, 캣아이, 크롬, 밀키/시럽/마이크로 프렌치, 나비/3D 파츠
- 웨딩 뷰티 트렌드: 자연스러운 피부, 소프트 글램, 피치/로즈/코랄, 로우번, 반묶음
- 반영구 시술 자료: 자연눈썹, 엠보, 콤보, 립블러셔, 점막 아이라인, 자연스러운 색감

## 카테고리

FINDY2의 핵심 카테고리는 아래 6개만 사용합니다.

- 헤어
- 네일아트
- 메이크업
- 포토
- 웨딩
- 반영구시술

왁싱은 FINDY2 기본 카테고리에 포함하지 않습니다.

## 데이터 구조

`BEAUTY_MOOD_KEYWORDS`는 카테고리별로 아래 그룹을 가집니다.

- `moods`: 세련된, 귀여운, 센치한, 드라마틱 같은 감성어
- `colors`: 블루블랙, 밀키화이트, 코랄, 필름톤 같은 색감/톤
- `length_or_shape`, `shot_types`, `hair_makeup`: 카테고리별 형태 정보
- `techniques`: 시술 방식 또는 표현 기법
- `concerns`: 사용자의 고민/특징
- `purposes`: 방문 목적이나 콘텐츠 목적
- `phrase_examples`: 실제 검색/추천 문장 예시

## 추천 알고리즘 사용 예

1. 사용자가 “차분한 애쉬블루 단발”을 입력합니다.
2. 앱은 `차분한`을 감성, `애쉬블루`를 색상, `단발`을 형태로 분리합니다.
3. 헤어 글/스냅/비디오 중 같은 태그를 가진 콘텐츠를 먼저 보여줍니다.
4. 사용자가 저장하거나 오래 보면 해당 태그의 가중치를 올립니다.
5. 나중에 FINDY 예약 서비스에서는 같은 태그를 잘 소화하는 샵/디자이너를 추천할 수 있습니다.

## 운영 기준

- 초기 사전은 넓게 가져가되 실제 추천 점수는 사용자 행동 데이터로 보정합니다.
- 유행어는 빠르게 바뀌므로 분기별로 `data/beauty_mood_keywords.py`를 갱신합니다.
- 감성어와 개인정보/비방 표현은 분리합니다. 리뷰 안전 필터는 `data/review_safety.py`에서 관리합니다.
- 카테고리명은 `헤어`, `네일아트`, `메이크업`, `포토`, `웨딩`, `반영구시술`로 정규화합니다.
