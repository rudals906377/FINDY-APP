# FINDY2

FINDY2는 첫 배포용 커뮤니티형 앱 기준 파일입니다. 고객과 아티스트를 나누지 않고 모두가 사용할 수 있는 뷰티 커뮤니티 앱으로 운영하며, 여기에서 모은 데이터를 바탕으로 나중에 FINDY의 추천/예약 시스템을 강화합니다.

## 실행

```bash
cd /Users/kyoungmin/Desktop/FINDY
python3 python_files/FINDY2.py
```

실제 앱 본문은 `python_files/FINDY2.py`에 있습니다.

## 현재 역할

- 첫 배포 기준 앱입니다.
- 예약 중심이 아니라 커뮤니티, 리뷰, 질문, 자유게시판, 스냅, 비디오 중심입니다.
- 아티스트 등록/운영 앱이 아니라 고객 중심 커뮤니티 앱입니다.
- 현재 FINDY2 앱 버전에서는 스냅/커뮤니티 작성 과정에 아티스트 선택이나 시술자 정보 노출을 두지 않습니다.
- FINDY와 디자인은 최대한 동일하게 유지하고, 기능만 커뮤니티 데이터 수집에 맞춥니다.

## 주요 기능

- 홈: 인기글, 리뷰, 질문, 자유게시판, FINDY 스냅, FINDY 비디오
- 하단 첫 탭: 카테고리 대신 커뮤니티로 운영
- 커뮤니티: 자유게시판, 리뷰, 질문만 한 화면에서 피드로 확인하고 스냅/비디오는 별도 하단 탭에서 운영
- 커뮤니티 글: 설정/내정보 화면과 같은 부드러운 라운드 카드, 연한 테두리, 태그형 배지 기준
- 커뮤니티/스냅 조작부: 작성, 분야, 정렬, 보기 전환처럼 같은 성격의 조작 정보는 한 줄 툴바로 압축
- 글 카드 하단 정보: 조회, 저장, 좋아요, 댓글은 본문과 얇은 구분선으로 분리하고 세로 여백을 최소화
- 글쓰기 화면: 커뮤니티, 스냅, 비디오 작성 페이지는 뒤로가기/완료 상단바, 한 줄 선택바, 큰 제목/내용 입력 영역, 하단 첨부 툴바 구조로 통일
- 공통 상단 액션바: FINDY 로고 옆에 현재 화면명(커뮤니티, 스냅, 비디오, 내정보 등)을 한 줄로 노출하고, 오른쪽에는 검색, 메시지 목록, 알림 버튼을 고정
- 검색 화면: 상단 돋보기 버튼을 누르면 전용 검색 레이어를 열고, 둥근 검색창, 사진 검색 준비 안내, 빠른 카테고리, FINDY 추천, 최근 검색어 삭제/재검색 동작을 제공
- 메인 탭 헤더: 본문 안 중복 헤더를 두지 않고 상단 공통바 하나를 기준점으로 사용
- 공통 레이아웃: 커뮤니티/내정보/검색/작성 등 일반 페이지는 좌우 여백을 줄여 콘텐츠 폭을 넓게 쓰고, 스냅/비디오 전용 화면은 기존 폭과 풀블리드 연출을 유지
- 화면 문구 정리: FINDY2 기본 동선에서는 아티스트/샵 중심 표현 대신 서비스, 방문 장소, 공개 표시명처럼 고객 커뮤니티에 맞는 중립 표현을 사용
- 버튼 피드백: 아직 연결 전인 기능은 `곧 연결될 예정` 톤으로 안내하고, 모든 클릭 피드백은 FINDY 메인 컬러 토스트로 통일
- 다중 선택 필터: 커뮤니티, 홈 카테고리 전용 페이지, 스냅, 비디오의 분야/정렬/보기 선택은 서랍형 선택창으로 제공하며 선택 상태를 유지
- 홈 카테고리 전용 페이지: 헤어, 네일아트, 메이크업, 포토, 웨딩, 반영구 분야와 전체, 리뷰, 질문, 자유게시판, 스냅, 비디오 콘텐츠를 서랍형으로 선택
- 홈 글 노출 순서: 인기글을 먼저 보여주고, 이후 리뷰, 질문, 자유게시판, 스냅, 비디오 순서로 고정
- 커뮤니티 글 정렬: 자유게시판, 리뷰, 질문 안에서 추천순, 최신순, 댓글순 중 사용자가 고른 정렬을 우선 적용
- 뷰티 분야: 헤어, 네일아트, 메이크업, 포토, 웨딩, 반영구시술 기준 유지
- 안전장치가 포함된 리뷰 작성/신고/상태 관리 로직
- 스냅/비디오 피드, 좋아요, 저장, 댓글, 공유, 신고, 팔로우 동작
- FINDY 비디오: 상단바 아래부터 여백 없이 풀블리드로 표시하고, 위/아래 슬라이드 시 다음/이전 영상 화면이 밀려 들어오는 전환감 적용
- 내정보: 내가 쓴글, 좋아요, 저장, 나의 FINDY, 설정, 프로필, 내 예약, 후기, 알림
- 내정보 서비스 연결: 내 예약은 예약내역, 후기는 내 후기, 알림은 공통 알림 페이지로 연결하며 쿠폰/이벤트는 준비 중 안내
- 나의 FINDY: 카테고리별 추천 정보 입력
- 화면 문구 안정화: 고객 커뮤니티 앱 기준에 맞춰 `나의 뷰티정보`, `시술`, `샵 정보`, `공개 프로필` 등 일부 구버전 표현을 `나의 FINDY`, `서비스/이용`, `방문 정보`, `공개 표시명` 중심으로 정리
- 기능 분리: FINDY2 현재 버전에서는 예약/직접 메시지를 비활성 플래그로 분리하고, 사용자 진입 시 준비 안내 화면으로 보냅니다. 예약/DM 코드는 추후 FINDY 본서비스에서 재활성화할 수 있도록 보관합니다.
- 알림 분리: 현재 알림은 커뮤니티 활동, 리뷰, 나의 FINDY 안내 중심으로 구성하고 예약 알림은 예약 플래그가 켜졌을 때만 합쳐집니다.
- 소식함: 상단 종이비행기 아이콘은 외부 DM이 아니라 FINDY 안내, 비디오 팁, 커뮤니티 활동 알림을 확인하는 `소식함`으로 운영합니다.
- 추천 키워드 사전: 헤어, 네일아트, 메이크업, 포토, 웨딩, 반영구시술별 감성어/색감/기법/고민/목적 데이터

## 수정 기준

- 새 커뮤니티 기능은 우선 FINDY2에 구현합니다.
- FINDY2에서 안정화된 디자인과 흐름은 추후 FINDY에 반영합니다.
- 앱 안 표기명은 `FINDY2`를 사용합니다. 파일명도 `python_files/FINDY2.py`를 기준으로 합니다.
- 왁싱, 샵추천, 아티스트 추천은 FINDY2 기본 범위에 포함하지 않습니다.
- 메시지는 외부 DM/샵 상담 기능이 아니라 앱 내부 안내와 비디오 팁 공유용 목록으로만 운영합니다.
