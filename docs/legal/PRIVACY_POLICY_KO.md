# FINDY2 개인정보 처리방침 초안

시행 예정일: 2026년 6월 25일  
상태: 베타 배포 준비용 초안

> 공개 배포 전 운영 주체의 정식 명칭, 주소, 이메일, 개인정보 보호 책임자 및 정책 공개 URL을 확정해야 합니다.

## 1. 처리하는 개인정보

FINDY2는 현재 다음 정보를 처리합니다.

- 계정: 이메일, 사용자 이름, 닉네임, 인증된 휴대폰 번호, 비밀번호 해시와 솔트
- 소셜 로그인: 네이버, 카카오, Google, Apple 공급자 사용자 ID, 검증 이메일, 공개 닉네임과 프로필 이미지
- 프로필: 공개 프로필과 뷰티 관심 분야, 스타일 및 추천 설정
- 사용자 콘텐츠: 글, 댓글, 리뷰, 신고 내용, 사진과 영상
- 활동: 좋아요, 저장, 팔로우, 재게시, 최근 검색어
- 앱 환경: 알림, 재생, 개인정보 및 접근성 설정

현재 비밀번호 원문은 저장하지 않으며 PBKDF2-SHA256 방식의 해시로 처리합니다.

## 2. 이용 목적

- 회원가입, 로그인과 세션 유지
- 휴대폰 인증, 아이디 찾기, 비밀번호 재설정과 부정 가입 방지
- 커뮤니티, 리뷰, 질문, 자유게시판, 스냅 및 비디오 기능 제공
- 콘텐츠 추천 기능 준비
- 신고 접수, 위험 콘텐츠 검토와 이용자 보호
- 오류 대응, 데이터 복구와 서비스 개선

## 3. 저장 위치와 제3자 제공

이메일 로그인 데이터, 활동, 콘텐츠와 업로드 파일은 현재 사용자의 기기에 로컬 저장합니다. 소셜 로그인 사용 시 FINDY 인증 게이트웨이와 선택한 로그인 공급자 사이에서 인증 코드, 토큰과 공개 프로필이 처리됩니다. 공급자 액세스 토큰과 Client Secret은 앱에 전달하지 않습니다.

향후 Supabase 등 외부 데이터 저장소, 분석, 알림 또는 동기화 서버를 연결하기 전에 처리방침과 스토어 데이터 공개 내용을 다시 갱신해야 합니다.

## 4. 보유 및 파기

정보는 사용자가 계정을 삭제하거나 앱 데이터를 제거할 때까지 보관됩니다. 앱의 `내정보 > 설정 > 계정 삭제`에서 계정을 삭제하면 다음 항목을 로컬 기기에서 삭제합니다.

- 회원 계정과 로그인 세션
- FINDY 인증 게이트웨이의 통합회원과 연결된 소셜 식별자
- 사용자별 글, 댓글, 리뷰, 반응과 설정
- 사용자가 업로드한 사진과 영상
- 마지막 정상 저장본 백업 파일

앱 삭제 시 운영체제 정책에 따라 앱 데이터가 제거될 수 있습니다.

## 5. 이용자의 권리

이용자는 앱에서 프로필과 추천 정보를 수정하고, 작성 콘텐츠를 삭제하고, 계정을 탈퇴할 수 있습니다. 공개 서비스 전 외부 계정 삭제 요청 URL과 문의 채널을 추가합니다.

## 6. 안전 조치

- 비밀번호 해시 저장
- 인증번호 단기 만료, 재발송 제한과 입력 시도 횟수 제한
- 인증번호 원문 미저장
- 사용자별 데이터 파일 분리
- 원자적 저장과 마지막 정상본 백업
- 업로드 파일 형식과 크기 검사
- 리뷰 개인정보 감지와 마스킹
- 신고 누적에 따른 숨김 및 관리자 검토 상태

## 7. 아동의 개인정보

연령 확인과 보호자 동의 정책이 아직 구현되지 않았습니다. 공개 배포 전 서비스 이용 가능 연령과 미성년자 처리 절차를 확정해야 합니다.
