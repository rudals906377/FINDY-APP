# FINDY2 스토어 데이터 공개 초안

기준일: 2026년 6월 25일  
대상 버전: FINDY2 0.1.0 로컬 베타

## 현재 구현 기준

| 데이터 범주 | 예시 | 수집/저장 | 외부 공유 | 목적 |
|---|---|---|---|---|
| 연락처 정보 | 이메일, 인증된 휴대폰 번호 | 기기 로컬 저장 및 인증 서버 처리 | 문자 발송/본인확인 공급자 연결 예정 | 계정 로그인과 복구 |
| 소셜 계정 정보 | 공급자 사용자 ID, 검증 이메일, 공개 프로필 | FINDY 인증 서버 처리 | 선택한 로그인 공급자 | 통합회원 로그인 |
| 사용자 ID | 사용자 이름, 닉네임 | 기기 로컬 저장 | 없음 | 프로필과 작성자 표시 |
| 사용자 콘텐츠 | 글, 댓글, 리뷰 | 기기 로컬 저장 | 없음 | 커뮤니티 기능 |
| 사진 및 동영상 | 첨부 미디어 | 기기 로컬 저장 | 없음 | 스냅, 비디오, 리뷰 |
| 앱 활동 | 좋아요, 저장, 검색어 | 기기 로컬 저장 | 없음 | 사용자 기능과 추천 준비 |
| 위치 정보 | 일시적 현재 위치, 선택 지역, 권한 상태, 갱신 시각 | 선택 지역/권한 상태 중심 로컬 저장 | 없음 | 내 주변 콘텐츠와 지역 기반 커뮤니티 |
| 환경 설정 | 알림, 재생 설정 | 기기 로컬 저장 | 없음 | 앱 환경 유지 |

## 현재 요청 권한

| 권한 | 사용 여부 | 사용 이유 |
|---|---|---|
| 사진 보관함 | 사용 | 글, 리뷰, 스냅에 사진 첨부 |
| 카메라 | 미사용 | 현재 직접 촬영 기능 없음 |
| 마이크 | 미사용 | 현재 녹음 기능 없음 |
| 위치 | 선택 사용 | 사용자가 내 주변 기능을 누를 때 현재 위치 기반 추천과 지역 기반 콘텐츠 표시 |

필요하지 않은 권한은 요청하지 않습니다.
위치 정보는 사용자가 직접 실행한 경우에만 요청하며 정확한 GPS 위치를 장기 보관하거나 백그라운드 위치 추적·위치 이력 저장에는 사용하지 않습니다.

## 공개 배포 전 필수 재검토

- Supabase/Auth/Storage 연결 시 데이터 전송과 보관 위치
- 푸시 알림 SDK의 기기 식별자 처리
- 분석·오류 수집 SDK 사용 여부
- 네이버, 카카오, Google, Apple 로그인 공급자의 데이터 처리와 정책 URL
- 문자 발송 및 PASS 본인확인 공급자의 데이터 처리, 위탁과 보유기간
- 외부 계정 삭제 요청 URL
- 운영자 연락처와 개인정보 처리방침 공개 URL
- 연령 등급과 미성년자 정책
