# FINDY2 Web

FINDY2의 커뮤니티 우선 첫 배포 방향을 브라우저에서 발표할 수 있도록 만든
HTML5, CSS3, JavaScript 기반 정적 웹사이트입니다.

## 실행

프로젝트 루트에서 아래 명령을 실행합니다.

```bash
python3 -m http.server 4173
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:4173/web/
```

`index.html` 파일을 직접 열어도 기본 기능은 동작하지만, 로컬 서버 실행을
권장합니다.

## 파일

- `index.html`: 시맨틱 마크업과 전체 화면 구조
- `styles.css`: FINDY2 브랜드 디자인과 반응형 레이아웃
- `app.js`: 화면 전환, 필터, 검색, 글쓰기, 좋아요/저장, 나의 FINDY 상태
- `report/`: 과제 보고서 PDF, 보고서 원문, 화면 캡처

## 주요 기능

- 커뮤니티: 자유, 리뷰, 질문 필터와 추천/최신/댓글순 정렬
- 스냅: 여섯 뷰티 분야 필터와 저장
- 비디오: 다음 팁 전환, 좋아요, 저장, 공유 문구 복사
- 글쓰기: 글 종류, 분야, 제목, 내용 입력 및 브라우저 저장
- 통합 검색과 최근 검색어
- 나의 FINDY: 분야별 분위기, 색감, 목적 저장
- 데스크톱 및 모바일 반응형 화면

## 저장 방식

서버나 데이터베이스 없이 `localStorage`를 사용합니다. 브라우저 개발자
도구에서 사이트 저장 데이터를 삭제하면 작성 글과 선택 상태가 초기화됩니다.
